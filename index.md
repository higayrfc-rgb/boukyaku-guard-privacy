# プライバシーポリシー（忘却がーど）

最終更新日：2026-02-18

「忘却がーど」（以下、「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

## 1. 収集する情報
本アプリは、現時点でユーザーの個人情報（氏名、メールアドレス、電話番号等）を収集しません。
また、本アプリはアカウント登録・ログイン機能を提供していません。

## 2. 端末内で扱うデータ
本アプリは、学習カード（単語・問題・メモ等）や学習履歴などのデータを、ユーザーの端末内（ローカルストレージ）に保存します。
これらのデータは、本アプリの学習機能提供のためにのみ使用され、開発者が取得することはありません。

## 3. 外部送信・第三者提供
本アプリは、現時点でユーザーのデータを外部に送信したり、第三者に提供したりしません。

## 4. 広告・解析について
本アプリは、現時点で広告配信SDKやアクセス解析ツールを使用していません。
将来的に広告を導入する場合、広告事業者による情報収集（端末識別子等）が行われる可能性があります。その際は、本ポリシーを更新し、必要に応じてアプリ内および/またはストア上で適切に案内します。
